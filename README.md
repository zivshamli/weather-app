# 🌦️ Weather App

A simple weather application that displays the weather based on the city name, using the [OpenWeatherMap API](https://openweathermap.org/).

## 🧩 Features
- 🔍 Search by city name to get the current weather.
- 🎤 Voice search support using Web Speech API.
- 🌡️ Displays temperature, humidity, and wind speed.
- 🌥️ Dynamic weather icons depending on conditions (Clouds, Clear, Rain, Drizzle, Mist).
- 🚫 Error message when city is not found.

## 📦 Technologies Used
- HTML, CSS, JavaScript
- [OpenWeatherMap API](https://openweathermap.org/api)
- Web Speech API for voice recognition

## 🎯 How to Use
1. Clone the repository.
2. Replace `YOUR_API_KEY` in the JavaScript file with your own OpenWeatherMap API key.
3. Open `index.html` in a browser.
4. Type or speak a city name to get the weather info.

---
## 🔊 Voice Search Instructions
1. Click the microphone button 🎤 next to the search bar.
2. Speak the name of the city clearly.
3. The app will automatically search and display the weather for the spoken city name.

> **Note**: Voice search works only in browsers that support the Web Speech API (like Chrome).
